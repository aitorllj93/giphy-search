Giphy Search

A GIPHY search client webapp

A running version of the application can be found here. There's also code documentation made with Compodoc. Test coverage reports are provided as well.

About this project

This project was created with the latest Angular version, and uses the Giphy API.

The application is built with Nx and uses Angular CLI, Jest and Cypress under the hood.

It's configured with pre-commit hooks (Husky) to automatically run code formatting (Prettier) and linting (ESLint) so you don't need to worry about it.

The User Interface is built with Bootstrap customized based on the GIPHY Brand Guidelines. The application is responsive and works on mobile, tablet and desktop. The Tag Input is built with ngx-chips component.

The application state is stored with the NGXS library.

Unit tests use ng-mocks and Spectator.

Development

Node.js and npm are required to install the application.

Installation

git clone git@github.com:d3v0ps/giphy-search.git
cd giphy-search
npm install
npm run prepare

Update your environment variables with the following:

export GIPHY_API_KEY=<your-api-key>

Run Development Server

npm start

Run Unit Tests

npm test

Run E2E Tests

npm run e2e
